# Multiple Choice Questions

## 1. Find the output I
Easy

**Problem Statement :** What is the output?
```
int main() {
    int x = 5;
    char y = 'A';

    y = x + y;
    cout << y;
}
```

- [ ] 70
- [ ] F
- [ ] 71
- [ ] E

## 2. Find the output II
Easy

**Problem Statement :** What is the output?
```
int main() {
    double a = 10.25;
    int b = (int) a/2;

    cout << b;
}
```

- [ ] 5.125
- [ ] 5
- [ ] 5.25
- [ ] 5.15

## Solutions
1. F
> ASCII value of A is 65, so 65+5=70 and F is the ASCII value of 70.

2. 5
> Converting a to int hence a=10 so b=5.
